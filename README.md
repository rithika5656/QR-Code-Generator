## How to Use

1. Enter a URL or text in the input field
2. Click "Generate QR Code" or press Enter
3. Click "Download QR Code" to save the image

## Technologies Used

- HTML5
- CSS3
- JavaScript
- [qrcode.js](https://github.com/davidshimjs/qrcodejs) library

## Setup

No installation required! Just clone the repository and open `index.html` in your browser.

```bash
git clone https://github.com/rithika5656/QR-Code-Generator.git
cd QR-Code-Generator
```

Then open `index.html` in your preferred browser.
